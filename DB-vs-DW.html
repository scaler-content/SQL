Database (DB)

The live, working engine — right now
  • ⚡ Used while you're ordering — real-time
  • 📝 Reads + writes every second
  • 📅 Keeps recent data (days / weeks)
  • 🎯 Optimised for speed, not analysis
  • 🍳 Like your kitchen — cooking right now

Data Warehouse (DW)

The historical archive — big picture
  • 📊 Used for reports, dashboards, analysis
  • 📖 Mostly reads — rarely writes
  • 🗓️ Stores years of historical data
  • 🔍 Optimised for complex queries at scale
  • 📚 Like a recipe book — every meal ever cooked

SQL is how you talk to the tables

Manager asks
How many orders did Bangalore get yesterday?
Your SQL
SELECT COUNT(*)
FROM zomato.orders
WHERE city = 'Bangalore'
AND order_date = '2024-01-15';
→ 2,847 orders
Manager asks
Which Prime members cancelled last month?
Your SQL
SELECT c.customer_id, c.city
FROM customers c
JOIN orders o
ON c.customer_id = o.customer_id
WHERE c.is_prime = 'Y'
AND o.status = 'cancelled';
→ 143 customers
